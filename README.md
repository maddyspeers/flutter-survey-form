# 📋 Flutter Survey Form (Standalone)

This is a standalone `main.dart` file for a Flutter survey form. It collects user input like name, graduation status, and number of classes using clean Material Design UI.

## 🔧 Tech Stack

- **Flutter SDK**
- **Dart**
- **Material Widgets**

## 📄 File Structure

lib/ └── main.dart

## 🚀 How to Use

To run this file in a Flutter environment:

1. Create a new Flutter project if you haven't already:
   ```bash
   flutter create survey_form
   cd survey_form
2. Replace the contents of lib/main.dart with this file:
   ```bash
    cp /path/to/your/main.dart lib/main.dart
3. Get dependencies:
    ```bash
    flutter pub get
3. Run the app:
    ```bash
    flutter run
## ✨ Features
- First and Last Name input
- Graduation status toggle (Checkbox)
- Class count slider (only if not graduating)
- Result screen or thank-you snackbar
- Clean and dynamic UI

## 📁 This file is ready to drop into any Flutter project. Just replace the default main.dart and you're good to go!
